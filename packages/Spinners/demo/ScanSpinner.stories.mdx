## ScanSpinner: Scanning for Awesomeness!

Whether you're building a search-centric app or just want to add a touch of exploration to your design, the ScanSpinner is here to turn your UI into a realm of possibilities.

### Features:

- **Dynamic Scanning**: The ScanSpinner starts its scanning journey as soon as it's placed, giving your UI an air of anticipation and dynamism.
- **Endless Exploration**: Like an explorer in uncharted territory, the ScanSpinner's animation loops endlessly, promising an ongoing adventure.
- **Seamless Integration**: Adding the ScanSpinner to your project is as easy as navigating an exciting quest, thanks to its user-friendly API.

### How to Use:

```jsx
import React from 'react';
import ScanSpinner from 'react-craftify-spinners';  

function App() {
  return (
    <div>
      <h1>Scanning for Awesomeness!</h1>
      <ScanSpinner style={{ width: "200px", height: "200px" }} />
    </div>
  );
}

```

### Custom backgroundColor:

```jsx
import React from 'react';
import ScanSpinner from 'react-craftify-spinners';  

function App() {
  const spinnerStyle = {
    width: '200px',
    height: '200px',
    backgroundColor: 'lightgray', 
  };

  return (
    <div style={{ display: 'flex', justifyContent: 'center', alignItems: 'center', height: '100vh' }}>
      <ScanSpinner style={spinnerStyle} />
    </div>
  );
}

export default App;
 
```
