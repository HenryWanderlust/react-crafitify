# Pulsing Loader

The `Pulsing` loader is a simple spinner that you can customize.


## Usage

To use the `Pulsing` loader, import it from `'../Pulsing'` and pass the `color` and `size` props to customize its appearance.

### Example:

```jsx
import Pulsing from '../Pulsing';

function MyComponent() {
  return (
    <div>
      <Pulsing color="#ff0000" size="30px" />
    </div>
  );
}
