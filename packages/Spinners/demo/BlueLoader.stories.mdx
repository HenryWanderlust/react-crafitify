# BlueLoader

BlueLoader is a spinner component that animates in a circular motion.

## Usage

To use the BlueLoader component, import it into your file and place it in your JSX/TSX where you want the loader to appear.

```jsx
import BluLoader from 'react-craftify-spinners';

const MyComponent = () => {
  return (
    <div>
      <BluLoader color="#004dff" size="56px" />
    </div>
  );
};
```
