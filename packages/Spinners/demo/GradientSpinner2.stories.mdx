## How GradientSpinner2 Works 🌀

The `GradientSpinner2` component is a visually appealing spinner that uses CSS gradients and animations to create a dynamic and captivating loading effect.

### Props ⚙️

- **size**: The size of the spinner. Accepts CSS size values (e.g., `"100px"`, `"2em"`).

- **startColor**: The starting color for the gradient. Accepts a CSS color value (e.g., `"#9b59b6"`).

- **midColor**: The middle color for the gradient. Accepts a CSS color value (e.g., `"#84cdfa"`).

- **endColor**: The ending color for the gradient. Accepts a CSS color value (e.g., `"#5ad1cd"`).

### How to Use 🛠️

To use the `GradientSpinner2`, import it from `'path/to/GradientSpinner2'` and include it in your component. You can customize the colors and size using the props mentioned above.

Example usage:

```jsx
import React from 'react';
import { GradientSpinner2 } from 'react-craftify-spinners';

const MyComponent = () => {
  return (
    <div>
      {/* Use the GradientSpinner2 component with custom props */}
      <GradientSpinner2
        startColor="#9b59b6"
        midColor="#84cdfa"
        endColor="#5ad1cd"
        size="96px"
      />
    </div>
  );
};
```
