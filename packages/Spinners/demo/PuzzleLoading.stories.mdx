### Features:

- **Dynamic Scanning**: The PuzzleLoading starts its scanning journey as soon as it's placed, giving your UI an air of anticipation and dynamism.
- **Endless Exploration**: Like an explorer in uncharted territory, the PuzzleLoading's animation loops endlessly, promising an ongoing adventure.
- **Seamless Integration**: Adding the PuzzleLoading to your project is as easy as navigating an exciting quest, thanks to its user-friendly API.

### How to Use:

```jsx
import React from 'react';
import PuzzleLoading from 'react-craftify-spinners;  

function App() {
  return (
    <div>
      <h1>Scanning for Awesomeness!</h1>
      <PuzzleLoading style={{ width: "200px", height: "200px" }} />
    </div>
  );
}

```

### Custom backgroundColor:
