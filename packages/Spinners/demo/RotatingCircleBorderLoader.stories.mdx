# Rotating Circle Border Loader

A customizable loader component with a rotating circle border animation.

## Usage

To use the `RotatingCircleBorderLoader` component, import it and render it in your React application.

```jsx
import React from 'react';
import RotatingCircleBorderLoader from 'react-craftify-spinners';

const MyComponent = () => {
  return (
    <div>
      <h1>Loading</h1>
      <RotatingCircleBorderLoader />
    </div>
  );
};

export default MyComponent;
```
