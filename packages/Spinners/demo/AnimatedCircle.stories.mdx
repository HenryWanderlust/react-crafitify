The `AnimatedCircle` component provides an animated SVG circle with customizable colors, size, and className.

## Basic Usage

You can use the `AnimatedCircle` like this:

```jsx
import React from 'react';
import {AnimatedCircle} from 'react-craftify-spinners';  

const MyComponent = () => {
  return (
    <div>
      <AnimatedCircle />
    </div>
  );
};
```

You can customize the color and size of the AnimatedCircle:
