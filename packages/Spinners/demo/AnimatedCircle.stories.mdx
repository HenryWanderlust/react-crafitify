# AnimatedCircle

The `AnimatedCircle` component provides an animated SVG circle with customizable colors, size, and className.

## Basic Usage

You can use the `AnimatedCircle` like this:

## Usage

```jsx
import React from 'react';
import AnimatedCircle from 'react-craftify-buttons';  

const MyComponent = () => {
  return (
    <div>
      <AnimatedCircle />
    </div>
  );
};
```


## Custom Color and Size


```jsx
import React from 'react';
import AnimatedCircle from 'react-craftify-buttons';  

const MyComponent = () => {
  return (
    <div>
      <AnimatedCircle color="blue" size="4em" />
    </div>
  );
};
```


## Custom Class Name

You can add a custom class name to the animated circle:

```jsx
import React from 'react';
import AnimatedCircle from 'react-craftify-buttons';  

const MyComponent = () => {
  return (
    <div>
      <AnimatedCircle className="my-custom-class" />
    </div>
  );
};
```
