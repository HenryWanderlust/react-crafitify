## FormsLoading: Mesmerizing FormsLoading! 🌀

Introducing the *FormsLoading* component, a captivating addition to your UI that brings mesmerizing forms to life.

Whether you're looking to add a touch of elegance to your design or want to create a dynamic loading animation, the FormsLoading component is here to infuse life into your projects. ✨

### Features:

- **Dynamic Animation**: 🌀 FormsLoading gracefully animates, creating an elegant and engaging visual experience.
- **Customization**: 📐 You can customize the width, height, and visibility of the FormsLoading component to suit your design needs.
- **Accessibility**: ♿ FormsLoading includes accessibility attributes to ensure a seamless user experience.

### How to Use:

```jsx
import React from 'react';
import {FormsLoading} from 'react-craftify-spinners';  

function App() {
  return (
    <div>
      {/* Use the FormsLoading component with custom props */}
      <FormsLoading style={{ width: '200px', height: '200px' }} />
    </div>
  );
}
