# Hypnotic Loader

The `Hypnotic` loader is a captivating spinner that you can customize.

 

## Usage

To use the `Hypnotic` loader, import it from `'react-craftify-spinners'` and pass the `size` prop to customize its size.

### Example:

```jsx
import Hypnotic from 'react-craftify-spinners';  

function MyComponent() {
  return (
    <div>
      <Hypnotic size="80px" />
    </div>
  );
}
