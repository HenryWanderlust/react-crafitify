# 🔄 Cube Loader: Spinning Fun for Your UI! 🔄

A cube loader component that displays a spinning cube animation.

## Usage 🚀

To use the `CubeLoader` component, import it and render it in your React application. You can customize the appearance of the cube loader using the following props:

- `color`: Change the color of the loader. 🎨
- `borderColor`: Change the color of the cube's border. 🌈
- `className`: Add a custom className to the component. 🆒

## Default Usage 🔃

<Canvas>
  <Story name="Default" args={{}} />
</Canvas>

```jsx
import React from 'react';
import CubeLoader from 'react-craftify-spinners';

const MyComponent = () => {
  return (
    <div>
      <h1>Loading</h1>
      <CubeLoader />
    </div>
  );
};
```

## Custom Color 🌈

You can change the color of the cube loader by using the color prop.

```jsx
import React from 'react';
import CubeLoader from 'react-craftify-spinners';

const MyComponent = () => {
  return (
    <div>
      <h1>Loading</h1>
      <CubeLoader color="#3498db" borderColor="#e74c3c" />
    </div>
  );
};

```

## Custom Border Color 🎨

You can change the border color of the cube by using the borderColor prop.

```jsx
import React from 'react';
import CubeLoader from 'react-craftify-spinners';

const MyComponent = () => {
  return (
    <div>
      <h1>Loading</h1>
      <CubeLoader borderColor="#ff0000" />
    </div>
  );
};

```

## Custom ClassName 🆒

You can add a custom className to the cube loader component by using the className prop.

```jsx
import React from 'react';
import CubeLoader from 'react-craftify-spinners';

const MyComponent = () => {
  return (
    <div>
      <h1>Loading</h1>
      <CubeLoader className="classname-loader" />
    </div>
  );
};

```


##  Custom Size 📏

You can also customize the size of the cube loader by using the size prop.

```jsx
import React from 'react';
import CubeLoader from 'react-craftify-spinners';

const MyComponent = () => {
  return (
    <div>
      <CubeLoader size="80px" />
    </div>
  );
};

```
