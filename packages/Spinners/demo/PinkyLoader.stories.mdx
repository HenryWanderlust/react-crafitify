## 🌸 PinkyLoader: Elegant Loading Animation

The PinkyLoader is a charming loading animation component designed to add a touch of elegance to your user interface. This delightful spinner brings a dynamic and engaging visual experience to your projects, making it perfect for creating captivating loading animations.

### Key Features:

- **Smooth Animation**: 🌀 PinkyLoader gracefully animates, creating an elegant and captivating loading experience.
- **Customizable Appearance**: 📐 You can customize the size, style, and appearance of the PinkyLoader component to match your design requirements.
- **Accessibility-Friendly**: ♿ PinkyLoader includes accessibility attributes, ensuring an inclusive user experience for all.

### How to Use:

To use the PinkyLoader in your React application, follow these steps:

1. Import the `PinkyLoader` component from 'react-craftify-spinners':

```jsx
import React from 'react';
import { PinkyLoader } from 'react-craftify-spinners';

function App() {
  return (
    <div>
      <PinkyLoader style={{ width: '200px', height: '200px' }} /> {/* Customize the style as needed */}
    </div>
  );
}

export default App;
