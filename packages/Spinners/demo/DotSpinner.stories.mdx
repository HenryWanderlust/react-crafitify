# DotSpinner

The `DotSpinner` component displays a spinning set of dots in a circular pattern.

## Usage

```jsx
import React from 'react';
import DotSpinner from 'react-craftify-spinners/DotSpinner';

const MyComponent = () => {
  return (
    <div>
      <h1>Loading</h1>
      <DotSpinner />
    </div>
  );
};
```
