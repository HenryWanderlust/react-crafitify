## Custom Color

You can customize the color of the spinner by using the `color` prop.

```jsx
import React from 'react';
import CircleLoader from 'react-craftify-spinners/CircleLoader';

const MyComponent = () => {
  return (
    <div>
      <h1>Loading</h1>
      <CircleLoader />
    </div>
  );
};

```
