# FancyArrowButton

The `FancyArrowButton` component is used for rendering a customizable button with fancy styling and gradient effects.

## Default

This section demonstrates the default rendering of the `FancyArrowButton` component.

```jsx
import React from 'react';
import FancyArrowButton from 'react-craft-buttons';

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <FancyArrowButton>Click Me</FancyArrowButton>
    </div>
  );
}; 
```
## Customized

This section demonstrates the rendering of a customized FancyArrowButton component with specific colors, sizes, and other properties.

```jsx
import React from 'react';
import FancyArrowButton from 'react-craft-buttons';

const MyComponent = () => {
  return (
    <FancyArrowButton
      size={180}
      mainBgColor="#814fd1"
      gradientBgColor1="rgb(222, 203, 226)"
      gradientBgColor2="rgb(79, 131, 209)"
      ringColor="#6f00ff"
      shadowColor="rgba(79, 99, 209, 0.64)"
      hoverShadowColor="rgba(79, 99, 209, 0.64)"
      mainFgColor="#313133"
      hoverFgColor="rgb(38, 11, 53)"
    >
      Customized Button
    </FancyArrowButton>
  );
};

 ```
