# CoinbaseButton

The `CoinbaseButton` component is a customizable button with the Coinbase button design.

## Usage

To use the `CoinbaseButton` component, import it and provide the necessary props.

```jsx
import React from 'react';
import AppleButton from  'react-craft-buttons';  

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <CoinbaseButton>Coinbase Button</CoinbaseButton>
    </div>
  );
};

```
## Properties

- `children`: Content of the button (string or JSX)
- `backgroundColor`: Background color of the button
- `borderColor`: Border color of the button
- `textColor`: Text color of the button
- `fontSize`: Font size of the button's text
- `width`: Width of the button
- `height`: Height of the button
- `disabled`: Boolean flag to disable the button
