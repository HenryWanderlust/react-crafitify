# AppleButton

The `AppleButton` component is used for rendering a customizable button with Apple-like styling.

## Default

This section demonstrates the default rendering of the `AppleButton` component.

```jsx
import React from 'react';
import AppleButton from  'react-craft-buttons';  

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <AppleButton>Conic Button</AppleButton>
    </div>
  );
};

```
## Disabled

This section demonstrates the rendering of a disabled AppleButton component.
