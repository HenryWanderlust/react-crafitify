# GenerateBtn

A button with a fancy shadow effect.

This is the GenerateBtn component. It allows you to customize various aspects of the button, such as its content, size, colors, and more.

## Usage

To use the `GenerateBtn` component :

```jsx
import React from 'react';
import GenerateBtn from 'react-craftify-buttons';

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <GenerateBtn>BUTTON</GenerateBtn>
    </div>
  );
};

```

 
## Disabled

import React from 'react';

## Customization

You can customize the GenerateBtn component using the following controls:

```jsx
import React from 'react';
import GenerateBtn from 'react-craftify-buttons';

const MyComponent = () => {
  return (
    <div>
       <GenerateBtn
        width="15em"
        height="5em"
        backgroundColor="#1C1A1C"
        hoverGradient="linear-gradient(0deg, #A47CF3, #683FEA)"
       >
          Customized Button
      </GenerateBtn>
    </div>
  );
};

```
