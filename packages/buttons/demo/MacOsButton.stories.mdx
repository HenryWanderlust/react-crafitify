# MacOsButton

The `MacOsButton` component is a customizable button with a Mac OS-like appearance.

## Usage

To use the `MacOsButton` component, import it and provide the desired props:

```jsx
import React from 'react';
import MacOsButton from  `react-craft-buttons`;  

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <MacOsButton>MacOs Button</MacOsButton>

      {/* Customized button with specific props */}
      <MacOsButton
        width={100}
        height={40}
        borderColor="#3A6CD9"
        borderWidth={2}
        boxShadow="0px 0.5px 1px rgba(0, 0, 0, 0.2)"
        backgroundColor="#EFEFEF"
        textColor="#555555"
      >
        Focus Me
      </MacOsButton>
    </div>
  );
};

export default MyComponent;
