# TiltedDiagonalButton

The `TiltedDiagonalButton` component provides a customizable tilted diagonal button with various options for customization.

## Basic Usage

You can use the `TiltedDiagonalButton` like this:

```jsx
import React from 'react';
import TiltedDiagonalButton from  `react-craftify-buttons`;  

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <TiltedDiagonalButton>MacOs Button</TiltedDiagonalButton>
    </div>
  );
};

export default MyComponent;

```
