# GradientButton

The `GradientButton` component is a customizable button with gradient background and various properties that can be adjusted.

## Basic Usage

Use the `GradientButton` component by passing your desired text as children.

```jsx
import React from 'react';
import GradientButton from 'react-craft-buttons';

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <GradientButton>Click me</GradientButton>
    </div>
  );
};
```

## Customization

Use the `GradientButton` component by passing your desired text as children.

```jsx
import React from 'react';
import GradientButton from 'react-craft-buttons';

const MyComponent = () => {
  return (
    <div>
     <GradientButton
      textColor="#ffffff"
      bgColor="#ff7f50"
      fontSize="20px"
      width="200px"
      height="50px"
    >
     Custom Button
    </GradientButton>
</div>
  );
};
```
