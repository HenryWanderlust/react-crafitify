# AmazonButton

The `AmazonButton` component is used for rendering a customizable button with Amazon-like styling.

## Default

This section demonstrates the default rendering of the `AmazonButton` component.

```jsx

{/* Default */}

<AmazonButton>
  Button 13
</AmazonButton> 

{/* Customized button with background color and text color */}

This section demonstrates a customized rendering of the AmazonButton component.

You can customize the appearance of the button using props.

<AmazonButton
  backgroundColor="#ff9900"
  borderColor="#ff9900"
  textColor="#ffffff"
  fontSize={16}
  width="150px"
>
  Custom Button
</AmazonButton>
