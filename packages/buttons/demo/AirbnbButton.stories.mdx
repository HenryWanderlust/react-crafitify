# AirbnbButton

The `AirbnbButton` component is used for rendering a customizable button with Airbnb-like styling.

## Default

This section demonstrates the default rendering of the `AirbnbButton` component.

```jsx
import React from 'react';
import AirbnbButton from 'react-craftify-buttons';

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <AirbnbButton>Airbnb Button</AirbnbButton>
    </div>
  );
};
```

## Disabled

This section demonstrates the rendering of a disabled AirbnbButton component.

```jsx
import React from 'react';
import AirbnbButton from 'react-craftify-buttons';

const MyComponent = () => {
  return (
    <AirbnbButton disabled>
      Disabled Airbnb Button
    </AirbnbButton>
  );
};
```


## Customized

This section demonstrates the rendering of a customized AirbnbButton component with specific background color, border color, hover background color, hover border color, text color, font size, width, height, border radius, and border width values.

```jsx
import React from 'react';
import AirbnbButton from 'react-craftify-buttons';


const MyComponent = () => {
  return (
    <AirbnbButton
      backgroundColor="#542dc9"
      borderColor="#222222"
      hoverBackgroundColor="#6d5cc5"
      hoverBorderColor="#333333"
      textColor="#000000"
      fontSize={15}
      width={150}
      height={50}
      borderRadius={10}
      borderWidth={2}
    >
      Customized Airbnb Button
    </AirbnbButton>
  );
};
```
