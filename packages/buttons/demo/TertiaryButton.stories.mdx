# TertiaryButton

A button component with a stylish tertiary style, designed for hover and focus effects.

## Usage

The `TertiaryButton` component offers a visually appealing button with dynamic hover and focus effects. To use it:

```jsx
import React from 'react';
import TertiaryButton from 'react-craft-buttons';

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <TertiaryButton>Stripe Button</TertiaryButton>
    </div>
  );
};
```
 

 ## Customized

The `TertiaryButton` component offers a visually appealing button with dynamic hover and focus effects. To use it:
