# CustomStyledButton

A custom styled button component.

 
## Usage

```jsx
import React from 'react';
import CustomStyledButton from  'react-craftify-buttons';  

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <CustomStyledButton>Default Button</CustomStyledButton>
    </div>
  );
};

```

## Primary Button

```jsx
import React from 'react';
import CustomStyledButton from  'react-craftify-buttons';

  return (
    <div>
```
