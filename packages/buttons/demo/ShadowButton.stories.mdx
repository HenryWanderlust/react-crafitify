# ShadowButton

The `ShadowButton` component is a customizable button with shadow effects and hover animations.

<CenteredStory>
  <ShadowButton>Shadow Button</ShadowButton>
</CenteredStory>

## Customization

You can customize the `ShadowButton` component using the following props:

- `textColor`: Color of the button text (default: `#000`)
- `border`: Border style of the button (default: `3px solid`)
- `hoverTextColor`: Color of the button text on hover (default: `#fff`)
- `hoverBorder`: Border style of the button on hover (default: `3px solid #38D2D2`)
- `hoverBackgroundColor`: Background color of the button on hover (default: `#38D2D2`)

```jsx
<ShadowButton
  textColor="#000"
  border="3px solid"
  hoverTextColor="#fff"
  hoverBorder="3px solid #38D2D2"
  hoverBackgroundColor="#38D2D2"
>
  Custom Shadow Button
</ShadowButton>
