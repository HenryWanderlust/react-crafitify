 # DiagonalSwipeButton

The `DiagonalSwipeButton` component is a customizable button with various options such as size, colors, and more.

## Usage

To use the `DiagonalSwipeButton` component, import it and provide the desired props:

```jsx
import React from 'react';
import DiagonalSwipeButton from 'react-craft-buttons';  

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <DiagonalSwipeButton>
         Hover Me! 
      </DiagonalSwipeButton>
    </div>
  );
};

export default MyComponent;
```

## Properties

-textColor (string): The color of the button text. Defaults to purple.

-borderColor (string): The color of the button border. Defaults to purple.

-borderRadius (string): The border radius of the button. Defaults to 0.

-hoverColor (string): The background color of the button on hover. Defaults to purple.

-width (string): The width of the button. Defaults to auto.

-height (string): The height of the button. Defaults to auto.

-fontSize (string): The font size of the button text. Defaults to 17px.
