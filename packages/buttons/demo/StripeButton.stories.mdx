# StripeButton

The `StripeButton` component is used for rendering a customizable button with Stripe-like styling.

## Default

This section demonstrates the default rendering of the `StripeButton` component.

```jsx
import React from 'react';
import StripeButton from 'react-craftify-buttons';

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <StripeButton>Stripe Button</StripeButton>
    </div>
  );
};
```

## Disabled

This section demonstrates the rendering of a disabled StripeButton component.

```jsx
import React from 'react';

const MyComponent = () => {

```



## Customized

This section demonstrates the rendering of a customized StripeButton component.

```jsx
import React from 'react';
import StripeButton from 'react-craftify-buttons';

const MyComponent = () => {
  return (
    <div>
      {/* Customized usage */}
      <StripeButton
        backgroundColor="#FF5733"
        boxShadow="rgba(0, 0, 0, 0.1) 0 1px 1px 0"
        textColor="#FFFFFF"
        fontSize={18}
        height={50}
        width={200}
      >
        Customized Stripe Button
      </StripeButton>
    </div>
  );
};
```
