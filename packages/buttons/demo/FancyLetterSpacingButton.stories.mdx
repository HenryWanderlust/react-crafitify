# FancyLetterSpacingButton

The `FancyLetterSpacingButton` component is used for rendering a customizable button with letter spacing effects.

## Default

This section demonstrates the default rendering of the `FancyLetterSpacingButton` component.

```jsx
import React from 'react';
import FancyLetterSpacingButton from 'react-craftify-buttons';

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <FancyLetterSpacingButton>BUTTON</FancyLetterSpacingButton>
    </div>
  );
};

```

## Customized

This section demonstrates the rendering of a customized FancyLetterSpacingButton component with specific colors, letter spacing values, sizes, and other properties.

```jsx
import React from 'react';
import FancyLetterSpacingButton from 'react-craftify-buttons';

const MyComponent = () => {
  return (
    <div>
    <FancyLetterSpacingButton
      color="black"
      hoverColor="white"
      activeColor="white"
      letterSpacing="0.2em"
      hoverLetterSpacing="0.5em"
      activeLetterSpacing="0.4em"
      size="medium"
      backgroundColor="transparent"
      hoverBackgroundColor="#171717"
    >
      Customized Button
    </FancyLetterSpacingButton>
    </div>
  );
};

```
