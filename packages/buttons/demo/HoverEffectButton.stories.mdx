# HoverEffectButton

A button with an appealing hover effect.

## Usage

The `HoverEffectButton` component provides a visually appealing hover effect on buttons. To use it:


## Usage

```jsx
import React from 'react';
import HoverEffectButton from 'react-craft-buttons';

const MyComponent = () => {
  return (
    <div>
       <HoverEffectButton>
        Custom Text
      </HoverEffectButton>
    </div>
  );
};
```
 
##  Customized button with hover effect 

```jsx
import React from 'react';
import HoverEffectButton from 'react-craft-buttons';

const MyComponent = () => {
  return (
    <div>
        <HoverEffectButton
        size="large"
        textColor="#ffffff"
        backgroundColor="#007bff"
        hoverColor="#0056b3"
      >
        Custom Text
      </HoverEffectButton>
    </div>
  );
};
```
