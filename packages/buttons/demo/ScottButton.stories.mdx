# ScottButton

The ScottButton component allows you to create customizable buttons with hover effects.

## Default

To use the ScottButton component, simply place it within your JSX code:

```jsx
import ScottButton from 'react-craft-buttons';  

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <ScottButton>Conic Button</ScottButton>
    </div>
  );
};
```

## Customized

```jsx
import ScottButton from 'react-craft-buttons';  

const MyComponent = () => {
  return (
    <div>
      <ScottButton
       borderColor="#FF5733"
       textColor="#FF5733"
       hoverBgColor="#333"
       hoverTextColor="white"
    >
      Customized Button
    </ScottButton>
   </div>
  );
};
```
