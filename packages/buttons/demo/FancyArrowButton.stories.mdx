# FancyArrowButton

The `FancyArrowButton` component is a button with customizable styles and arrow animations.

## Usage

To use the `FancyArrowButton` component, import it and provide the desired props:

```jsx
import React from 'react';
import FancyArrowButton from 'react-craftify-buttons';  

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <FancyArrowButton>Default Button</FancyArrowButton>

      {/* Customized button */}
      <FancyArrowButton
        mainBgColor="#FF5733"
        arrowColor="#FFF"
        width="200px"
        height="60px"
      >
        Custom Button
      </FancyArrowButton>
    </div>
  );
};

export default MyComponent;
```

## Props

- `size` (optional): The size of the button. Can be a string representing any custom size, e.g., 'small', 'medium', 'large', 'custom'.
- `mainBgColor` (optional): The background color of the button.
- `arrowColor` (optional): The color of the arrow icon.
- `width` (optional): The width of the button.
- `height` (optional): The height of the button.
