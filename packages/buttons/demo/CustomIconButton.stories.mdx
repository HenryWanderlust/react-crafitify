# CustomIconButton

The `CustomIconButton` component is a customizable button with various options such as size, colors, and more.

## Usage

To use the `CustomIconButton` component, import it and provide the desired props:

```jsx
import React from 'react';
import CustomIconButton from 'react-craftify-buttons';  

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <CustomIconButton>
         Hover Me! 
      </CustomIconButton>
    </div>
  );
};

export default MyComponent;
```


## Properties

- `size: Size of the button ('small', 'medium', 'large')
- `backgroundColor: Background color of the button
- `textColor: Text color of the button
- `hoverColor: Hover color of the button
- `activeColor: Active color of the button
- `width: Width of the button
- `height: Height of the button


## Large Size Button


```jsx
import React from 'react';
import CustomIconButton from 'react-craftify-buttons';  
const MyComponent = () => {
  return (
    <div>
      {/* Large size button */}
      <CustomIconButton
        size="large"
        backgroundColor="#6a92ef"
        textColor="white"
        hoverColor="hsl(217, 77%, 71%)"
        activeColor="hsl(218, 76%, 84%)"
        width="200px"
        height="60px"
      >
        Hover Me!
      </CustomIconButton>
    </div>
  );
};

export default MyComponent;
```
