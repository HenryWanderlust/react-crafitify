# SecondaryButton

A secondary button component with customizable styles.

## Usage

The `SecondaryButton` component provides a secondary button with customizable styling options. To use it:

```jsx
import SecondaryButton from 'react-craft-buttons';

const MyComponent = () => {
  return (
    <SecondaryButton
      onClick={() => console.log('Button clicked')}
      disabled={false}
    >
      Secondary Button
    </SecondaryButton>
  );
};
