# Gradient Border Button

The `GradientBorderButton` is a customizable button component with gradient border effects and hover animations.

## Default
 
 ```jsx
import React from 'react';
import GradientBorderButton from 'react-craft-buttons';

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <GradientBorderButton>Click Me</GradientBorderButton>
    </div>
  );
}; 
```


## Custom Colors & Hover

 ```jsx
import React from 'react';
import GradientBorderButton from 'react-craft-buttons';

const MyComponent = () => {
  return (
    <div>
       <GradientBorderButton
         borderColor="#e63946"
         color="#e63946"
         hoverBackgroundColor="radial-gradient(#e63946, #f38181)"
         hoverBoxShadow="0px 0px 0.3rem #e63946"
        >
          Custom Colors & Hover
       </GradientBorderButton>
    </div>
  );
}; 
```


## Large Size

 ```jsx
import React from 'react';
import GradientBorderButton from 'react-craft-buttons';

const MyComponent = () => {
  return (
    <div>
        <GradientBorderButton size="large">
           Large Button
        </GradientBorderButton>
    </div>
  );
}; 
```
