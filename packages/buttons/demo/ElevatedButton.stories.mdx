# ElevatedButton

The `ElevatedButton` component is a customizable button with elevated styles and hover effects.

## Usage

To use the `ElevatedButton` component, import it from the appropriate location and provide the desired props:

```jsx
import React from 'react';
import ElevatedButton from 'react-craftify-buttons';  

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <ElevatedButton>Default Button</ElevatedButton>

      {/* Customized button with various properties */}
      <ElevatedButton
        backgroundColor="#FF5733"
        textColor="#FFF"
        fontSize="16px"
        hoverEffect={true}
        boxShadowColor="rgba(0, 0, 0, 0.2)"   
      >
        Custom Button
      </ElevatedButton>
    </div>
  );
};

export default MyComponent;
