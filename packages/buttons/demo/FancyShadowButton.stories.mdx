# FancyShadowButton

A button with a fancy shadow design.

## Usage

To use the `FancyShadowButton` component, import it and include it in your JSX:

```jsx
import React from 'react';
import FancyShadowButton from 'react-craftify-buttons';

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <FancyShadowButton>BUTTON</FancyShadowButton>
    </div>
  );
};

```
## Props

The FancyShadowButton component accepts the following props:

-`bgColor` (string, optional): The background color of the button. Default: #68ecc0.
-`children` (ReactNode, required): The content of the button.

## Custom Background Color

You can customize the background color of the button:

```jsx
import React from 'react';
import FancyShadowButton from 'react-craftify-buttons';

const MyComponent = () => {
  return (
    <div>
       <FancyShadowButton bgColor="#0074D9">
           Blue Button
       </FancyShadowButton>
    </div>
  );
};

```
