# Button51

The `Button51` component from the `react-craft-buttons` library is a customizable button with a pseudo-element background.

## Usage

To use the `Button51` component, import it from the `react-craft-buttons` package and provide the desired props:

```jsx
import React from 'react';
import Button51 from 'react-craft-buttons';  

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <Button51>Default Button</Button51>

      {/* Customized button with various props */}
      <Button51
        backgroundColor="#FC6C85"
        borderColor="#FF5733"
        textColor="#FFFFFF"
        pseudoBackgroundColor="#FFD1DC"
        hoverPseudoBackgroundColor="#ef66d8"
        fontSize="20px"
        height="40px"
        fontWeight={100}
      >
        Customized Button
      </Button51>
    </div>
  );
};

export default MyComponent;
