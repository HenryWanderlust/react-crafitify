# LoveButton

The `LoveButton` component is a customizable button with a heart animation.

## Usage

To use the `LoveButton` component, import it and provide the desired props:

```jsx
import React from 'react';
import Lovebtn from 'react-craftify-buttons';

const MyComponent = () => {
  return (
    <div>
      {/* Default usage */}
      <Lovebtn>Hover Me!</Lovebtn>

      {/* Customized button with different sizes */}
      <Lovebtn size="small">Small Button</Lovebtn>
      <Lovebtn size="medium">Medium Button</Lovebtn>
      <Lovebtn size="large">Large Button</Lovebtn>
    </div>
  );
};

export default MyComponent;
